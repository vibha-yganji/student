Python Syntax Examples

Lists

Python lists are ordered collections of items. They can contain elements of different data types.

Creating a List

        
my_list = [1, 2, 3, 'four', 5.0]

Accessing List Elements

        
# Access the first element (index 0)
first_element = my_list[0]

# Access the last element (index -1)
last_element = my_list[-1]

Modifying a List

        
# Append an element to the end
my_list.append(6)

# Insert an element at a specific index
my_list.insert(3, 'new_element')

# Remove an element by value
my_list.remove(3)

# Remove an element by index
del my_list[1]

Dictionaries

Python dictionaries are collections of key-value pairs. Each key is unique and maps to a value.

Creating a Dictionary

        
my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}

Accessing Dictionary Values

        
# Access value by key
name = my_dict['name']

# Using get() method with a default value
country = my_dict.get('country', 'USA')

Modifying a Dictionary

        
# Adding a new key-value pair
my_dict['job'] = 'Engineer'

# Modifying an existing value
my_dict['age'] = 31

# Removing a key-value pair
del my_dict['city']
